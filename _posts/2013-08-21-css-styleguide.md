## <a id='coding-style'></a>Coding-style
* Use soft-tabs with a two space indent.
* Keep lines fewer than 80 characters.
* Never leave trailing whitespace.

* Alphabetically order rules.

```
h1 {
  color: #222;
  font-size: 2em;
  font-weight: 500;
  line-height: 1.5;
  margin: 1em 0;
}
```
